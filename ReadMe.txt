Navigate to the "src" folder to find the java program files.

FirstQ.java -  program to find minimum of 4 numbers.
SecondQ.java - program to print odd numbers from 1 to 100 .
ThirdQ.java -  program to find whether the given number is prime or not.
FourthQ.java - program to print reverse of the given number.
FifthQ.java -  program to find the given number is Armstrong or not .
SixthQ.java -  program to find the given number is palindrome or not.
SeventhQ.java - program to print the factorial of a given number.
EigthQ.java -  program to find the Fibonacci series up to given number.
NinthQ.java -  program to find whether the given is perfect number or not.
